This is the info page to assist in the development of the elavare app.

To run the app:
- flutter emulators
- find device emulator you want to run
- flutter emulators --launch Medium_Phone_API_36.1 for example
- after it starts up:
   - flutter run to build and deploy the app to the emulated device
